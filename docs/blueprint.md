# **App Name**: SizeWise Poker

## Core Features:

- Room Creation: Host can create a room with customizable story sizing methods.
- Story Submission: Host can add stories for team members to vote on.
- Team Member Voting: Team members can join the room via a link or ID and vote on stories using cards. Includes option to add comments to votes.
- Card Reveal: Host can reveal the cards to display sizing estimations from all team members.
- Points Distribution: A donut chart displays the distribution of points with an average shown in the center. Use an AI tool to summarize this visualization and surface important information about group consensus and points that were outliers.
- History Presentation: Host can see, present, and share the history of stories, sizes, and votes given in the room.

## Style Guidelines:

- Primary color: Strong violet (#9466FF) to convey intellectual precision, the center of considered discussion.
- Background color: Very light lavender (#F5F0FF). Visibly in the same hue as the primary, but heavily desaturated.
- Accent color: Teal (#66FFDA). To the 'left' of the primary color in hue; strongly contrasting with the primary and background.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; for headlines and body text
- Use clean, professional icons to represent different story point values and voting actions.
- A clean and structured layout, that can quickly display vote distributions.
- Subtle transitions and animations when revealing cards and updating the donut chart.
